# Claude Code Setup Guide

> **English | [中文](README.zh-CN.md)**

Pre-class setup instructions for the **MSEM Module (20 March 2026)**. This guide walks students through installing and configuring Claude Code on macOS or Windows in 10-15 minutes.

## Contents

| File | Description |
|------|-------------|
| `claude-code-install-guide.md` | Step-by-step installation guide for macOS and Windows |
| `settings.json` | Example API configuration file for `~/.claude/settings.json` |
| `.claude.json` | Onboarding skip configuration for `~/.claude.json` |

## Quick Start

1. Follow the instructions in [`claude-code-install-guide.md`](claude-code-install-guide.md) for your operating system
2. Replace `YOUR_API_KEY` in the settings file with the key provided during class
3. Open a new terminal window and run `claude` to verify everything works

## Prerequisites

- macOS or Windows
- Stable internet connection
- 4 GB+ RAM
- Git for Windows (Windows only)

## Troubleshooting

See the [Troubleshooting section](claude-code-install-guide.md#troubleshooting) in the installation guide for common issues and fixes.
