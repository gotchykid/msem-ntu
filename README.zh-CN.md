# Claude Code 安装指南

> **[English](README.md) | 中文**

MSEM 模块（2026年3月20日）课前准备说明。本指南帮助学生在 10-15 分钟内完成 Claude Code 在 macOS 或 Windows 上的安装和配置。

## 目录

| 文件 | 说明 |
|------|------|
| `claude-code-install-guide.zh-CN.md` | macOS 和 Windows 的分步安装指南 |
| `settings.json` | `~/.claude/settings.json` 的 API 配置示例文件 |
| `.claude.json` | `~/.claude.json` 的跳过引导配置 |

## 快速开始

1. 根据你的操作系统，按照 [`claude-code-install-guide.zh-CN.md`](claude-code-install-guide.zh-CN.md) 中的说明操作
2. 将配置文件中的 `YOUR_API_KEY` 替换为课堂上提供的密钥
3. 打开新的终端窗口，运行 `claude` 验证是否一切正常

## 前置条件

- macOS 或 Windows
- 稳定的网络连接
- 4 GB 以上内存
- Git for Windows（仅 Windows 需要）

## 故障排除

请参阅安装指南中的[故障排除部分](claude-code-install-guide.zh-CN.md#故障排除)，了解常见问题和解决方法。
